# Capability: foreign-developer-credibility-signals

## Purpose
외국인 개발자 전용 페이지의 신뢰 근거 노출 기준을 정의한다.

## ADDED Requirements

### Requirement: Foreign-developer page MUST show credibility signals
The system MUST include credibility signals on `/for-foreign-developers/` based on verifiable career and network background.

#### Scenario: Credibility block is rendered
- **WHEN** a user requests `/for-foreign-developers/`
- **THEN** credibility signals include Samsung experience and regional context
- **AND** network-based support positioning is displayed
