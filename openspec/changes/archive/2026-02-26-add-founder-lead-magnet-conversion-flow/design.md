## Context

현재 메인 랜딩은 유료/상담 문의 중심이며, 첫 방문자가 바로 유료 상담으로 진입하기 어려운 구조다. 기존 시스템은 `ContactInquiry` 저장, SMTP 메일 발송, `FunnelEvent` 기반 내부 전환 이벤트 집계, 관리자 대시보드까지 이미 갖춰져 있다. 따라서 신규 리드마그넷은 기존 문의/메일/대시보드 체계를 확장하는 방식이 구현 리스크가 가장 낮다.

핵심 제약은 다음과 같다.
- 기존 문의 플로우를 깨지 않고 메인 랜딩에서만 리드마그넷 진입을 제공해야 한다.
- 진단 결과는 즉시 일부 제공하고, 전체 자료는 이메일 입력 이후 전달되어야 한다.
- 운영 관점에서 리드마그넷 리드와 일반 문의를 분리 조회할 수 있어야 한다.

## Goals / Non-Goals

**Goals:**
- 메인 랜딩에 `3분/8문항` 무료 AX 진단 흐름을 추가한다.
- 점수 기반 A/B/C 결과 분류와 등급별 CTA 브릿지를 제공한다.
- 리드마그넷 전용 이메일 전달(전체 리포트/체크리스트)과 내부 저장을 구현한다.
- 관리자 화면에서 리드마그넷 리드를 일반 문의와 분리해 확인 가능하게 한다.

**Non-Goals:**
- 결제/예약 시스템 추가
- GA4 기반 신규 이벤트 설계
- 외국인 개발자 페이지의 리드마그넷 동일 적용

## Decisions

### Decision 1: 리드마그넷은 문의 폼과 분리된 전용 폼/엔드포인트로 구현한다
- 전용 `LeadMagnetForm`과 `lead_magnet_submit` 엔드포인트를 추가한다.
- 이유: 문의 목적과 진단 목적의 필드/검증/응답 구조가 다르며, 분리 시 유지보수와 테스트가 명확해진다.
- 대안: 기존 `ContactForm` 확장. 기각 이유: 조건 분기 과다로 폼 복잡도 급증.

### Decision 2: 결과 저장은 기존 ContactInquiry를 재사용한다
- 리드마그넷 제출도 `ContactInquiry`에 `inquiry_type=lead_magnet_diagnosis`로 저장하고, 메시지에 진단 요약(점수/등급/Top 5)을 구조화해 기록한다.
- 이유: 운영 조회/메일 실패 재처리/기존 관리자 워크플로 재사용 가능.
- 대안: 별도 모델 신설. 기각 이유: 초기 범위 대비 운영 복잡도 증가.

### Decision 3: 전환 측정은 FunnelEvent 확장으로 처리한다
- `lead_magnet_start`, `lead_magnet_submit`, `lead_magnet_email_sent` 이벤트를 내부적으로 기록한다.
- 이유: 이미 대시보드에서 이벤트 기반 집계가 동작 중이며 외부 도구 의존을 늘리지 않는다.

### Decision 4: 등급 브릿지는 점수 구간 고정 규칙으로 단순화한다
- A: 생산성 개선 상담
- B: AX 구축 상담
- C: 외주 집중 트랙 상담(동시 1개사 정책 안내)
- 이유: 운영 메시지 일관성과 구현 단순성 확보.

## Risks / Trade-offs

- [Risk] 이메일 미수신/오입력으로 리포트 전달 누락 → Mitigation: 제출 성공 후 내부 저장 우선, 발송 실패 상태 기록 및 관리자 재발송 기능 활용
- [Risk] 진단 질문 품질이 낮으면 리드 품질 저하 → Mitigation: 질문/가중치 상수화 + 추후 A/B 테스트 가능 구조
- [Risk] 메인 랜딩 정보량 증가로 집중도 저하 → Mitigation: 리드마그넷 섹션을 접이형/요약형으로 배치하고 CTA를 명확히 분리

## Migration Plan

1. 폼/뷰/템플릿에 리드마그넷 제출 흐름을 추가한다.
2. `ContactInquiry.inquiry_type`에 리드마그넷 유형을 반영하고 저장 포맷을 고정한다.
3. 메일러에 리드마그넷 결과 전달 템플릿을 추가한다.
4. 관리자 대시보드 필터/통계를 리드마그넷 기준으로 확장한다.
5. 테스트(`lead_magnet submit`, `email sent/failure`, `dashboard segmentation`)를 추가하고 실행한다.

Rollback:
- 리드마그넷 라우트/섹션 제거 + 신규 분기 로직 제거로 기존 문의 플로우로 즉시 복귀 가능.
