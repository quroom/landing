## 1) 타겟 및 메시지
- 주요 타겟 고객:
  - 메인(`/`): 아이디어는 있지만 제품화 경험이 부족한 예비 창업자/사업자
  - 전용(`/for-foreign-developers/`): 한국에서 개발 커리어를 시작하려는 외국인 개발자 및 연계 파트너
- 고객의 핵심 문제:
  - 아이디어를 실제 서비스로 만들 실행 파트너가 부족함
  - 개발/출시/수익화까지의 우선순위를 정하지 못함
  - 외국인 개발자의 실무 연계와 커뮤니케이션 구조가 부족함
- 우리 서비스의 핵심 해결책:
  - 메인: AX 진단/AX 구축/외주용역 트랙으로 창업자 실행 속도와 성과를 개선
  - 전용: 외국인 개발자 실무/취업 연계, 온보딩, 협업 구조 지원
- 차별점(경쟁사 대비 3가지):
  1. 단순 외주 개발이 아니라 사업성과 연결된 제품 개발 중심
  2. 개발 구현 + 운영/개선 + 수익화 관점을 하나의 흐름으로 제공
  3. 외국인 개발자 협업 경험을 기반으로 실무 연계 체계를 지원
- 브랜드 톤앤매너: 전문적, 신뢰감, 실행 중심, 명확한 커뮤니케이션
