## MODIFIED Requirements

### Requirement: Founder pages MUST present AX execution services
The system MUST present AX services for founders using founder-value language that prioritizes business problems, execution speed, and accountable outcomes over internal tool or methodology names.

#### Scenario: AX services are visible for founder journey
- **WHEN** a user views `/` or `/for-founders/`
- **THEN** AX execution services are listed with founder-facing outcomes
- **AND** each service maps to a consultation CTA
- **AND** service summaries explain what founder problem is addressed before mentioning internal execution tools or approaches
- **AND** service bullets use direct wording that clarifies what gets reviewed, fixed, built, or handed over
- **AND** service wording makes it clear which work can actually be delegated now, not only which strategic topic can be discussed
